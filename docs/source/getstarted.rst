Getting Started
===============

Installation
------------

.. code-block:: shell

    pip install casioplot


Requirements
------------

This module use python 3.11.

Need the module :py:mod:`Pillow` to work.

Install :py:mod:`Pillow`:
~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: shell

    pip install pillow
